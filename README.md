# mini-tf - a tiny Infrastructure-as-Code engine

### Intro
`mini-tf` is designed to be a basic Terraform clone from scratch to learn Golang. I wanted something useful and real to start working with it.

As my plan anyways was to commit changes to the official Terraform repository including providers etc. I find this being a good starting point for me to learn.

### Plan
I'll design mini-tf to be created in "rungs", with each rung being a small part of a bigger, continously evolving project. I'll start basic and continue to add advanced features in the future as I expand my knowledge in working with Golang.
